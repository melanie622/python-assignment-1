Python Assignment – Smart Home & EV Analysis
📌 Overview
This repository contains the full implementation of the assignment using Python. The primary implementation is provided as a Jupyter Notebook, with an additional Python script version included as a fallback.
📁 Files Included
notebook.ipynb – Main implementation (recommended)
script.py – Alternative Python script version
caseA_smart_home_30min_summer.csv – Dataset 1
caseA_ev_events.csv – Dataset 2
▶️ How to Run the Project
Option 1: Run using Jupyter Notebook (Recommended)
Open the notebook (.ipynb) in Google Colab
Upload the required datasets:
caseA_smart_home_30min_summer.csv
caseA_ev_events.csv
Run all cells sequentially to reproduce the results
Option 2: Run using Python Script
Open Google Colab
Copy the contents of script.py into a new Colab cell
Upload the required datasets
Run the code
⚙️ Requirements
No additional package installation is required.
The code runs using the default environment provided by Google Colab.
📌 Notes
Ensure both datasets are uploaded before running the code
